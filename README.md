# Portafolio Profesional - Antonio Lira

Este es un portafolio web minimalista y de alta precisión diseñado para mostrar proyectos de comunicación, diseño y estrategia.

## Tecnologías
- HTML5 / CSS3 (Vanilla)
- JavaScript (Vanilla)
- Lucide Icons (CDN)
- Google Fonts (Prompt)

## Estructura de Archivos
- `index.html`: Estructura principal.
- `style.css`: Sistema de diseño y animaciones.
- `portfolio.js`: Lógica de navegación y modales.
- `config.js`: Configuración dinámica de proyectos e imágenes.
- `Fotos/`: Directorio que contiene todos los activos visuales.

## Despliegue en Vercel
1. Sube este repositorio a GitHub.
2. Conecta tu cuenta de Vercel a GitHub.
3. Selecciona este repositorio y dale a 'Deploy'.
